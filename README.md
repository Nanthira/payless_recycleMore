# WasteLess Fee & Map

WasteLess helps users estimate Bangkok waste collection fees, build a daily waste list, and explore reported Bangkok district waste data on a map.

## Run

```text
npm start
```

Open `http://127.0.0.1:4173/`.

## Test

```text
npm test
```

## Project Structure

- `docs/`: project notes, domain rules, design, and testing guidance.
- `src/`: app code, generated Bangkok waste map data, raw source data, and local scripts.
- `tests/`: automated calculator tests.
